ScoreCraft iPhone修正版

修正内容
1. マイクラブ保存処理の不具合を修正
   - クラブIDの参照ミスを修正
   - 保存後に内容を再読込して確認
   - 保存完了後は設定画面へ戻る

2. 設定した入力モードを新規スコア入力へ反映
   - 下書きが残っていても、設定画面の最新モードを反映
   - 入力済みのスコアなどは維持

3. ゴルフ場保存後に設定画面へ戻る

4. PWAキャッシュを更新

GitHub更新後、iPhoneで古い表示が残る場合
- ホーム画面のScoreCraftを終了して開き直す
- それでも変わらない場合はSafariでURLを開いて再読み込みする
