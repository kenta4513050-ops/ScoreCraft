ScoreCraft Step10 完成版
========================

今回の追加・最終調整
- PWA対応（ホーム画面への追加・アプリ風起動）
- オフラインキャッシュ対応
- アプリアイコン・manifest追加
- iPhoneのホーム画面追加案内
- Android/PCのインストールボタン対応
- ノッチ・ホームインジケーターのセーフエリア対応
- タップ操作・入力欄のスマホ最適化
- 日付取得を端末のローカル日付に修正
- UUID非対応環境向けフォールバック追加
- 既存のStep1〜9機能をすべて維持

動作確認
1. ScoreCraftフォルダ内のindex.htmlを開きます。
2. 通常機能はファイルを直接開いて確認できます。
3. インストール・オフライン機能はHTTPSまたはlocalhostで有効です。
4. 設定画面の「スマホアプリとして使用」に案内が表示されます。

注意
ブラウザのlocalStorageにデータを保存します。
端末変更やブラウザデータ削除に備え、設定画面から定期的にバックアップしてください。
