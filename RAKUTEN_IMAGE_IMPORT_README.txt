ScoreCraft Ver1.2.0 楽天スコア画像取込版

追加・変更内容
- 楽天ゴルフスコア管理アプリの横長スコアカード画像を1～2枚選択して解析
- 同じラウンドの左右画像を統合
- 読み取り結果を1～18番ホールごとに確認・修正
- ラウンド履歴へ保存
- 未登録コースは18ホールのPAR情報をゴルフ場管理へ同時登録
- 重複ラウンド警告
- CSVインポート機能を削除
- 設定画面の「スマホアプリとして使用」欄を削除

注意
- OCRにはTesseract.jsを使用するため、画像解析時のみインターネット接続が必要です。
- 画像レイアウトや端末表示倍率により誤認識する場合があるため、保存前の確認・修正画面を必ず表示します。
