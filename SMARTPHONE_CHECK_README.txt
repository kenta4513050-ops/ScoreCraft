ScoreCraft スマホ確認手順
============================

【一番簡単な方法：PCとスマホを同じWi-Fiにつなぐ】

1. ZIPを展開します。
2. WindowsでScoreCraftフォルダを開き、フォルダのアドレス欄に「cmd」と入力してEnterを押します。
3. 黒い画面で次を入力します。

   py -m http.server 8000 --bind 0.0.0.0

   「py」が使えない場合：
   python -m http.server 8000 --bind 0.0.0.0

4. 別のコマンドプロンプトで「ipconfig」を実行し、Wi-FiのIPv4アドレスを確認します。
   例：192.168.1.20
5. スマホのブラウザで次を開きます。

   http://192.168.1.20:8000/

6. 確認後は、サーバーを起動した黒い画面で Ctrl+C を押して終了します。

【注意】
・PCとスマホは同じWi-Fiに接続してください。
・Windowsファイアウォールの確認が出た場合は、プライベートネットワークのみ許可してください。
・スマホごとにlocalStorageが別なので、PC側の登録データは自動では引き継がれません。バックアップ／復元で移行できます。
・ホーム画面追加や完全なPWAインストールは、原則としてHTTPSで公開した場合に有効です。HTTPのローカルIPでは通常のブラウザ動作確認が中心です。
