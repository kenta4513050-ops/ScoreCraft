ScoreCraft Ver1.2.1 楽天画像読取修正版

【主な修正】
・画像全体への汎用OCRから、楽天スコアカードの固定レイアウト解析へ変更
・緑色の「ホール」行と表の縦罫線を検出し、各列をホール番号へ対応付け
・各行を次の意味として個別に読み取るよう変更
  Par：各ホールのPAR
  プレイヤー名の行：スコア
  パット：パット数
  FWキープ・1on：ティーショット方向（丸印は中央・キープ・1on）
  ティーショット：使用クラブ
  バンカー：バンカー数
  OB：OB数
  ペナルティ：1ペナ数
・横スクロールで分かれた同一ラウンドの画像を最大2枚まで統合
・未登録コースは18ホールのPARが揃った場合にゴルフ場管理へ登録
・PWAキャッシュ名を更新

【注意】
画像認識には誤認識の可能性があります。保存前の確認画面で必ず修正してください。
楽天アプリ側の画面デザインが変更された場合は、再調整が必要になることがあります。
